Convert raw data to useful information (real-time data -> efficiency, reduce costs, predicts risks)

## Basic Features (Delivered by AI)
- AI Forecasted KPIs
- Risks prediction and alert
- Optimization stimulation
- Interactive AI assistant
- The 'Burnout Radar'
- Active risk forecast
- Weather Forecast
- The maintanence scheduler


## Value
- Prevent accidents before it happens and suggest the ways to overcome
- Bridges the gab between complex technical operations and strategic business management
- Turns raw data into profit or minimize the loss

## Usage
- Install any browser
- Download both index.html and script.js files and put inside a new folder.
- Find the index.html and open that file once both index.html and script.js files are in a same folder.
- The webpage will run on the browser
